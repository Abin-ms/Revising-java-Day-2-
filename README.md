## 5. Shallow Copy Concept

Understood the drawback of reference copying and learned about shallow copy.

### File

* `note.md`

### Concepts

* Shared memory reference
* Changes affecting both arrays
* Shallow copy behavior

### Explanation

When one array value changes, the copied array also changes because both arrays point to the same memory location.
